## Installation

1. Clone the repo
   ```sh
   git clone 
   ```

2. change to task 2
   ```sh
   cd task2
   ```

3. Install NPM packages
   ```sh
   npm install
   ```

4. Start the Application
   ```sh
   npm start
   ```
