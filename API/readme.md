## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rdhss/Backend-Test
   ```

2. change to API
   ```sh
   cd API
   ```

3. Install NPM packages
   ```sh
   npm install
   ```

4. Start the Application
   ```sh
   npm start
   ```
